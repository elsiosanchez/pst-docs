# Fase VII · ii. Evidencias del proyecto

## ¿Qué pide el esquema?

Un **apartado de evidencias** que sustente con documentos verificables lo declarado en la Matriz de Seguimiento y Evaluación. Aparece al final de esa matriz, pero en la práctica funciona como una sección propia.

::: danger No es opcional
Para muchos estudiantes que arrancan en el PST I, este apartado pasa desapercibido. **Sin evidencias, el proyecto no puede sustentarse en la defensa.** Cada actividad declarada en el cronograma y en la matriz debe tener su documento de respaldo.
:::

## Los tres tipos de documentos obligatorios

### 1. Fotografías de cada sesión o actividad

Registro visual del trabajo realizado en campo.

**Reglas básicas:**

- Agrupadas **por fecha o por sesión** (no como un álbum caótico).
- Cada foto con **título descriptivo** ("Tercera visita — Continuación de LibreOffice Calc: formato condicional").
- Deben aparecer **tanto los estudiantes facilitadores** como los **miembros de la comunidad** participando — no solo paisajes o pantallas.
- Resolución suficiente para que se distingan personas y materiales (evitar capturas borrosas o muy oscuras).

::: tip Buena práctica
Una foto vertical de pantalla de celular con título superpuesto (ej.: *"TERCERA VISITA — Recopilación de formatos básicos"*) sirve perfectamente como portada de la sesión en el apartado de evidencias.
:::

### 2. Actas de cada visita

Documento oficial que valida la visita ante la comunidad y la institución.

**Campos mínimos de cada acta:**

| Campo | Qué debe ir |
| --- | --- |
| **Fecha** | Día, mes y año de la visita |
| **Lugar** | Sede o espacio donde se realizó |
| **Actividad realizada** | Tema o taller específico |
| **Lista de asistentes** | Nombre completo · C.I. · Cargo · Firma |
| **Sello de la institución comunitaria** | **Obligatorio** para validar la visita |
| **Firma del responsable de la actividad** | Estudiante facilitador a cargo |

::: warning El sello no es decorativo
Sin el **sello de la institución comunitaria**, el acta no tiene validez ante la coordinación. Asegúrate de llevar el formato impreso a cada visita y pedir el sello en el mismo encuentro — no después.
:::

#### Modelo mínimo de acta

```text
ACTA Nº ___

En la ciudad de ____________________, a los ____ días del mes de
____________________ del año ________, siendo las ____:____ horas, en la
sede de ____________________________________, se realizó la actividad
"___________________________________________________________________"
en el marco del Proyecto Socio Tecnológico I — PNFI / UPTP.

ASISTENTES:

| Nº | Nombre completo | C.I. | Cargo | Firma |
|----|-----------------|------|-------|-------|
| 1  |                 |      |       |       |
| 2  |                 |      |       |       |
| 3  |                 |      |       |       |
| …  |                 |      |       |       |

Sin más por agregar, se cierra la presente acta a las ____:____ horas.

___________________________          ___________________________
Responsable de la actividad           Sello de la institución
(estudiante facilitador)              comunitaria
```

### 3. Control de Visitas PST

Formato institucional **único** donde se registran **todas** las visitas del proyecto en orden cronológico. Es el documento que conecta el cronograma planificado (Fase V) con la ejecución real (Fase VI).

**Cada fila del control de visitas incluye:**

- **Fecha** de la visita.
- **Aspectos tratados** (resumen breve de la actividad).
- **Asistentes** (nombres y firmas de los participantes ese día).
- **Firma del responsable** estudiantil de la sesión.

::: tip Una sola hoja, varias visitas
El Control de Visitas PST se completa **a mano, sesión por sesión**, en el mismo formato institucional. Al cierre del proyecto, esa hoja (o esas pocas hojas) muestra de un vistazo todo el recorrido del equipo en la comunidad.
:::

## Cómo organizar el apartado en el informe

Sugerencia de orden interno del apartado de Evidencias:

1. **Control de Visitas PST** (visión cronológica completa).
2. **Actas firmadas y selladas**, en orden cronológico.
3. **Fotografías agrupadas por sesión**, con su título descriptivo.

Si entregas el informe en digital, los anexos pueden ir como PDF escaneado de cada acta + un álbum de fotos por carpeta de sesión.

## Recomendaciones

- **Lleva siempre el formato impreso** a la visita. Conseguir firmas y sello *después* es difícil y a veces imposible.
- **No edites las fotos** más allá de recortes y rotación. Filtros artísticos restan formalidad.
- **Numera correlativamente** las actas (Acta Nº 1, Nº 2…). Si una visita se cae, no rompas la numeración — anota la fecha real.
- **Haz dos copias** del Control de Visitas: una para la comunidad, otra para tu archivo.
- Si una visita **no pudo realizarse**, déjalo registrado con la causa. La honestidad documental se valora más que un cronograma "perfecto".
