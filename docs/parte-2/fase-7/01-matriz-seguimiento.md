# Fase VII · i. Matriz de Seguimiento y Evaluación

## ¿Qué pide el esquema?

Una **matriz** que permita evaluar el cumplimiento del proyecto. Las columnas típicas son:

| Columna | Qué va |
| --- | --- |
| **Resultado esperado** | Qué se busca lograr |
| **Indicador** | Qué evidencia confirma el logro |
| **Ponderación** | % de peso de ese resultado en el total (suman 100%) |
| **Actividades** | Qué se hace para alcanzarlo |
| **Tiempo (inicio/fin)** | Fechas |
| **Medio de verificación** | Cómo se prueba el cumplimiento |
| **Complejidad** | Corto / Mediano / Largo plazo |
| **Responsable** | Quién del equipo lo lidera |

::: tip Verificación rápida
La columna **Ponderación** debe sumar **100%** exactamente. En el ejemplo: 10 + 10 + 20 + 20 + 20 + 20 = **100** ✓
:::

## Recomendaciones

- Los **medios de verificación** deben ser **tangibles**: planes de trabajo entregados, fotografías, listas de asistencia, archivos digitales producidos, capturas de pantalla.
- Distribuir **responsabilidades** entre los integrantes del equipo: nadie debe figurar como responsable de todo, ni nadie debe quedar sin responsabilidades.
- Si un resultado **no se cumplió al 100%**, dejá registro de cuánto sí y cuánto no. La honestidad académica se valora.
