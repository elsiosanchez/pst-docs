# Fase VI · ii. Alfabetización tecnológica

## ¿Qué pide el esquema?

Tres elementos:

1. **Objetivo general y específicos** de la alfabetización tecnológica.
2. **Plan diario de clase** — planificación de cada encuentro.
3. **Guía didáctica de apoyo** a los encuentros (manuales/cartillas para los participantes).

## Ejemplo de referencia

### Objetivo General

> Capacitar tecnológicamente a los miembros de la Sub-Comisión Regional de Árbitros en el uso eficiente de herramientas ofimáticas (Hoja de cálculo, Procesador de texto y Presentaciones) para optimizar la gestión administrativa y arbitral.

### Objetivos Específicos

1. Diagnosticar el nivel de conocimientos previos de los participantes mediante un instrumento exploratorio.
2. Instruir en el uso de Microsoft Excel / LibreOffice Calc para la creación de bases de datos de árbitros y estadísticas de partidos.
3. Enseñar el uso de Microsoft Word / Writer para la redacción estandarizada de informes arbitrales y cartas administrativas.
4. Capacitar en Canva y PowerPoint / Impress para la creación de material visual de apoyo en las charlas técnicas sobre las reglas del juego.

### Guía didáctica de apoyo

El proyecto incluye guías escritas para cada herramienta enseñada:

- **LibreOffice Calc** — interfaz, filas/columnas/celdas, operaciones básicas, formato condicional, tablas dinámicas, funciones SUMA() y SI().
- **LibreOffice Writer** — interfaz, formato de texto, tablas y listas, elementos gráficos, configuración de página, guardado.
- **LibreOffice Impress** — pestañas principales, diseño de diapositivas, transiciones y animaciones, ejemplos prácticos.
- **Canva** — creación de cuenta, interfaz, plantillas, personalización, elementos gráficos, exportación.

## Recomendaciones

- El **plan diario** se vuelve más fuerte si se referencia al cuestionario de diagnóstico inicial (qué nivel detectaste y cómo eso definió los contenidos).
- Las **guías didácticas** deberían entregarse **impresas o digitales** a los participantes, no quedarse solo en el documento final del PST.
- Pensá ejemplos pertinentes a la comunidad: en el ejemplo, se usa "tarjetas amarillas, faltas por equipo" para los ejercicios de Excel — eso es **alfabetización contextualizada**, no genérica.
- Documentá con **fotos** y **listas de asistencia** cada encuentro (van en Evidencias).
