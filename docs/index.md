## ¿Para quién es esta guía?

Para estudiantes del **Programa Nacional de Formación en Informática (PNFI)** que estén elaborando su **Proyecto Socio Tecnológico I**, sus tutores, y cualquier integrante de un equipo de proyecto que necesite entender qué pide cada apartado del esquema y cómo se ve un ejemplo bien hecho.

## ¿Qué encontrarás?

| Sección | Qué contiene |
| --- | --- |
| **Parte I — Abordaje y diagnóstico** | Descripción del proceso, metodología empleada y reflexión crítica del abordaje. |
| **Parte II — Planificación (EML)** | Las 7 fases del Enfoque del Marco Lógico, cada una con sus puntos. |
| **Reflexión sobre el EML** | Cierre crítico sobre la aplicación del Enfoque del Marco Lógico. |
| **Recursos** | Glosario, referencias bibliográficas y plantillas/herramientas sugeridas. |

## ¿Cómo usar la guía?

1. **Leé la sección** que vas a desarrollar.
2. **Revisá lo que pide el esquema** (qué se espera formalmente).
3. **Mirá el ejemplo de referencia** para entender el tono, la profundidad y la estructura.
4. **Aplicalo a tu propia comunidad** — no copies, adaptá.

::: info Tip rápido
Si estás recién arrancando, lee primero la **[Introducción](/introduccion)** y después seguí el orden del menú lateral. No te saltes fases — el EML es como una receta de cocina: el orden importa.
:::
